## Border Radius

| Token | Value | Usage |
|---|---|---|
| `--radius-sm` | 2px | Subtle rounding on inputs |
| `--radius-md` | 4px | Cards, buttons |
| `--radius-none` | 0 | Default — the brand aesthetic is clean/architectural |

The brand aesthetic calls for **minimal rounding** — sharp corners are more editorial and refined for a culinary school of this tone.

---

## Shadows

Minimal. Use sparingly:
- `--shadow-subtle`: `0 2px 16px rgba(28, 26, 23, 0.06)` — for floating elements
- `--shadow-card`: `0 4px 32px rgba(28, 26, 23, 0.08)` — for menu cards

---

## Design Principles from Brand Image

1. **Clean whitespace** — the cream background should breathe
2. **Typography-first** — Cormorant Garamond's elegance carries the design
3. **Gold as a signal** — accent color used sparingly as a mark of quality
4. **No decorative clutter** — the brand is confident, not busy
